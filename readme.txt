Modulatron:

To run this program, the .xcodeproj file must be opened in the latest version of Xcode and run with the iPhone 6.0 Simulator.

There does not need to be any extra sound packages installed.

Run the program - you can press the start button. The program should begin progressing through major key chord changes with 
C as the root. 

Use the options below to change the key and to also select a chord to modulate to. 

The program will calculate how many modulation steps are necessary and modulate to the destination key
over time. Once it hits the key, it will continue to progress through changes on that key. 

press stop button to stop sound.
